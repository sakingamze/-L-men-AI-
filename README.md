# Lümen-AI
Lümen-AI, yapay zekâ destekli bir kod mentorudur.  Python, JavaScript, Java ve C# gibi popüler programlama dillerindeki kodlarınızı analiz eder ve size hızlı, net öneriler sunar. 

Özellikler:
- Hataları tespit eder: Syntax, mantık, performans ve güvenlik (SQL injection, XSS)
- Kod kalite puanı verir (1-10)
- Kısa veya detaylı öneri modu
- Kod refactor ve performans iyileştirme önerileri
- Modern, göz yormayan gri tonlu arayüz
- Çoklu dil algılama

- Kullanımı:
1. Kodunuzu sol sütuna yapıştırın
2. Mentör profilinizi ve öneri modunu seçin
3. “Analiz Et” butonuna basın
4. Sağ sütunda analiz ve öneriler görünür

Bu uygulama Streamlit ile geliştirilmiştir ve GitHub’da barındırılmaktadır.
